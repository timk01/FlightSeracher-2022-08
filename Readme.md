Телеграм-бот, который помогает планировать путешествия. Сервис позволяет запрашивать данные к ценам по авиабилетам,
сохранять данные по выбранным направлениям. Также сервис позволяет добавлять событие перелета для конкретного
пользователя и учитывать уже существующие перелеты. В проекте используется travelpayouts pubilc API.
